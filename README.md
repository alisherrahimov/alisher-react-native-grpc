# @alisher/react-native-grpc

grpc library for react-native

## Installation

```sh
npm install @alisher/react-native-grpc
```

## Usage

```js
import { multiply } from '@alisher/react-native-grpc';

// ...

const result = await multiply(3, 7);
```
